Examples

Forms

Form submission is not considered as a navigation so it hard reloads the page by default. We have to consider using AJAX validation and the Core.redirect(href, ContextualTransition) method if we want to redirect our users to a Thank you page. ContextualTransition can also be set to false to utilize the view transition. Try this out by hovering on the button below for 1.5 seconds to be taken to the API Reference page, using a contextual transition (more on that below).

Third-party Scripts

Depending on our websites we might have to deal with third-party scripts used by plugins or others. Those scripts have to be evaluated on each page transition in order to be runned again.

Unfortunately there is a danger to evaluate malicious code and we cannot take the responsability of potential issues related to that. That's why we are never going to implement any evaluation method in Highway.

Check out this Github issue to have more informations about this topic.
